外勤工時計算使用說明

使用步驟：

一、選擇班別，請參考114年1月前幾日上班日期選擇。

二、輸入時薪，時薪為每小時加班費，並非俸點。

三、選擇要查詢年、月份。

四、輸入請假、停休、公假等資料。

五、查看計算後工時及加班費情形。

　

欄位說明：

外勤加休：114年2月起每月加休為2天，欄位起始預設為「2」。

超勤補休：須扣之前累積時數12小時，並當月返還工時12小時。

休假：包括休、婚、喪等公務人員假別，當月返還工時8小時。

上班改輪休：調班後，原上班日變輪休。

不扣時數外宿：原上班日改8時~20時上班，中間須休息2小時，當日工時為10小時。

停休整日：原輪休日改整日上班，包含調班。

停休改外宿：原輪休日改8時至18時上班，當日工時為10小時，若中間有排休息時間，應使用「其他停休時數」欄位。

其他停休時數：原輪休日停休增加工時，以小時計算。輸入負數可以減少工時。

公假：輸入公假開始日期及結束日期，如有跨月份，僅計算當月日期。最多可登打3段公假。

公假計算方式：公假區間遇公務員上班日(星期一~五(扣除國定假日)、補班日)，每日記工時8小時。

　　　　　　　公假區間遇公務員假日(星期六、日、國定假日)，為輪休，不記工時。

　　　　　　　如遇颱風等特殊情況，或勤務有特殊規定，有增減工時，請自行計算於「其他停休時數」。

調整週休二日計算：將日期區間調整為星期一~星期五上班，星期六、日為輪休，不扣除國定假日。

　　　　　　　　　依照人事室規定，請假區間達七日以上，應修改為週休二日(勤五休二)，輪休、加休、全部超勤補休可以排除。

　　　　　　　　　如改為週休二日，加休、補休、休假天數加總要達到"需請假天數"。

　

其他資訊說明：

基本工時：公務人員本月工時。

本月工時：外勤人員經計算後本月工時。

報滿加班費時數：領滿加班費所需工時(19000/時薪)。

累積工時：本月可累積的超勤時數(本月工時-基本工時-報滿加班費時數)。

可領加班費：上限19000元，須先上滿基本工時才可領取。

基礎工作日：本月星期一~星期五天數。

國定假日：人事行政局公告之假日數，包含補假，但不包含在星期六、日之假日。每年底更新。

補上班日：人事行政局公告之補上班之日數。每年底更新。

停班日：人事行政局公告因防災等因素，臺南市原上班改放假之日數，發生後進行更新。

原上班：外勤人員當月輪班，原本為上班之日數。

實際上班：外勤人員勤休經過調整後，當月實際上整天班之日數。

外宿：外勤人員勤休經過調整後，當月實際外宿之日數。
